##Introduction

Grammatical type: noun feminine.  
Occurrences:  1x HB (0/0/1); 0x Sir; 0x Qum; 0x Inscr.  (Total: 1x).

* Ketubim: Prov 23:29.


## 7. Conclusion

<b>A.1</b> <span dir="rtl">חכלילות</span> stands for a ‘dark’ colour, possibly ‘dark red’. The only surviving 
occurrence of this lexeme is in a description of eyes affected by the consumption of an abundance of 
wine (Prov 23:29-30).
